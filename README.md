# Django/React Note APP

Este é um projeto que utiliza Django para o backend e Vite com React para o frontend.

## Estrutura do Projeto

O projeto está organizado da seguinte maneira:

- **backend/**: Contém o código do servidor Django.
- **frontend/**: Contém o código do cliente React com Vite.

## Configuração do Backend (Django)

### Requisitos

Certifique-se de ter o Python instalado.

### Instalação

```bash
cd backend
pip install -r requirements.txt
```

### Execução do Servidor

```bash
python manage.py runserver
```

O servidor estará em execução em http://localhost:8000.

## Configuração do Frontend (Vite com React)
### Requisitos

### Certifique-se de ter Node.js e npm instalados.
### Instalação

```bash
cd frontend
npm install
```

### Execução do Cliente
```bash
npm run dev
```

O aplicativo estará em execução em http://localhost:5173
